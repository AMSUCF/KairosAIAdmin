Design
ACH 2025 Edition 2.0:
For the 2025 Association for Computers and the Humanities Conference, this game has been reimagined through the lens of Digital Humanities center administration. In this new iteration, players take on the role of a DH center director navigating challenges at the intersection of computational methods and humanities scholarship engaging generative AI. The scenarios have been rewritten to reflect contemporary DH concerns: environmental impacts of GPU computing, the ethics of web scraping and data collection, AI-generated historical "discoveries," and the perpetual challenge of explaining APIs to administrators.

Kairos Edition:
Designed to be displayed alongside the original, this updated version was created for Kairos using new art assests (generated with DALL-E 3, in some cases using the previous version as reference images). The code was updated to reflect the new color palette and mouseover zones using Claude Sonnet 4.0, working in Agent mode within GitHub Copilot.
Generation
For Version 2.0, much of the original game was maintained, but the content was rewritten through a series of prompts to Claude Opus 4.0 requesting a digital humanities perspective. The emphasis is now on the tensions between supporting creative and impactful digital work, and the very real problems raised by the tools themselves (and the companies pushing their rapid adoption in higher education). This iteration still fully reflects the generated voice.
